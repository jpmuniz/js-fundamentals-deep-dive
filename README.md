JavaScript é uma linguagem que pode ser executada tanto no browser quanto no servidor.

Para isso, ela depende de um runtime, como navegadores ou o Node.js, que fornece as APIs necessárias para interação com o ambiente.

No browser, o JavaScript interage com o HTML por meio da DOM (Document Object Model).
Através da tag <script>, o código JavaScript tem acesso ao objeto document, permitindo manipular elementos, reagir a eventos e controlar o comportamento da aplicação conforme os requisitos.

JavaScript foi projetado como uma linguagem single-thread, ou seja, executa um frame/função por vez.

Para lidar com operações assíncronas sem bloquear a execução, o runtime utiliza o Event Loop, que coordena a ordem de execução das tarefas.
